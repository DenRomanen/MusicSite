# Track List Table

Обновлен интерфейс списка песен на главной странице без изменений backend-логики.

- `frontend/src/widgets/track-library/TrackLibrary.tsx` переведен с карточной сетки на компактный tabular list, добавлены realtime-поиск по названию и сортировка А-Я / Я-А поверх уже отфильтрованного списка
- `frontend/src/entities/track/ui/TrackListRow.tsx` добавлен новый компонент строки списка с сохранением текущего аудиоплеера и удаления только своих треков
- `frontend/src/app/styles/index.css` обновлены стили списка, тулбара поиска/сортировки и адаптивного поведения для десктопа и планшета
