# Mobile Track Player Row

Скорректировано мобильное отображение строки списка треков.

- В `frontend/src/app/styles/index.css` для breakpoint `max-width: 720px` скрыты ячейки исполнителя и даты
- Название трека сохранено в верхней строке мобильной карточки
- Плеер остается отдельной строкой под названием и растягивается на доступную ширину
- Desktop и tablet-отображение не изменены
