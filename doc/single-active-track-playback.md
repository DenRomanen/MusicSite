# Single Active Track Playback

Сделано ограничение на одновременное воспроизведение только одного трека в списке.

- В `frontend/src/widgets/track-library/TrackLibrary.tsx` добавлена централизованная координация активного `audio` через `ref`, чтобы запуск нового трека останавливал предыдущий и сбрасывал его в начало
- В `frontend/src/entities/track/ui/TrackListRow.tsx` прокинуты события `onPlay`, `onPause` и `onEnded` из встроенного аудиоплеера в родительский список
- При удалении текущего воспроизводимого трека воспроизведение теперь останавливается до удаления элемента из списка
