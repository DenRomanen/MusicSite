# Mobile Track Title Wrap

Исправлено отображение названий песен в мобильной версии списка треков.

- В `frontend/src/app/styles/index.css` сохранено однострочное поведение на desktop
- Для breakpoint `max-width: 720px` заголовок трека переведен на перенос с ограничением до 2 строк через `-webkit-line-clamp`
- Убрано мобильное принудительное поведение одной строки для названия, добавлен `overflow-wrap: anywhere`
- Скорректированы `min-height`, `padding` и вертикальное выравнивание строки списка, чтобы длинный заголовок не наезжал на соседние элементы
