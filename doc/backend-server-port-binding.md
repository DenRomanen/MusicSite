# Backend Server Port Binding

Обновлен запуск backend-сервера под облачный деплой и внешнее пробрасывание порта.

- В `backend/src/server.ts` запуск сервера переведен на `const port = process.env.PORT || 10000`
- `app.listen` оставлен на `0.0.0.0`, чтобы сервер принимал внешние подключения в контейнере и на хостинге
- Лог старта синхронизирован с текущим портом: `Server started on ${port}`
